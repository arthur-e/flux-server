About this Visualization
This visualization demonstrates the spatial aggregation of carbon fluxes. The minimum monthly (or daily) carbon flux from the model is assigned to each state at the approriate time step. Using the minimum value (or maximum value) provides a stable representation of the aggregated fluxes that is not sensitive to the sample size (the number of modeled flux values within the state's boundaries). Both the color and the size of the state represent the flux value.

Flux values above the population mean correspond with an increased size of the state (the state is "breathing out" more carbon to the atmosphere). Flux values below the population mean correspond with a decreased size of the state (the state is "inhaling" more carbon from the atmosphere). States with a monthly (or daily) minimum flux value close to the mean appear white in color and take on their normal size with respect to the basemap.
